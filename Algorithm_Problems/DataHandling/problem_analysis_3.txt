문제 분석

3. 동적 메모리 할당의 크기 제한을 고려해야 한다

상황
malloc(sizeof(int) * n);
(1 <= n <= 3,000,000)

malloc이 작동 가능한 n의 최대값이 3,000,000 보다 작은 m이라면
(3,000,000 / m)개의 덩어리chunk로 나누어 처리해야 한다

예
m: 100,000
덩어리의 개수: 3,000,000 / 100,000
덩어리들: m1, m2, m3... m30 

3-2 buffer를 사용하지 않고 문제를 해결한다 <- 채택

데이터를 읽어들임과 동시에 
데이터의 연산을 누적해간다.
*/
